## ❓ よくある質問（FAQ）

### Q. 通知が届きません
A. 設定アプリで通知が許可されているかご確認ください。また、価格が希望価格を下回っている必要があります。

### Q. 商品が正しく登録できません
A. Amazon.co.jpの商品URLを使用してください。他国のAmazonには対応していません。
